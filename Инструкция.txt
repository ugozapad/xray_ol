X-Ray build 1098 (3 February 2002)

Учтите, что кроме xr_3da я ничего не портировал.

Требуется: Visual Studio 2013 вместе с Windows SDK 8.1 (он вроде ставится вместе с ним)
1. Открыть файл solution xr_3da\XR_3DA.sln
2. В меню студий Build -> Build Solution
3. Бинарники появятся в корневой папке папке binaries

Для запуска скачайте билд 1098. Можете заменить там бинарники или же запустить бинарники со студий указав рабочий каталог и прописав аргумент -server lev5_aztec

При дебаге будьте осторожны - Windows 10 может сойти с ума, из-за чего мышка начнёт дико тупить.
